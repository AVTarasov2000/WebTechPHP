Техническое задание.

Цель проекта- создать приложение для отслеживания затрат и доходов семьи, отслеживая затраты и доходы разных ее членов.

Задачи к решению:

  • Создание базы данных пользвателей
  • Создание регистрационной формы
  • Шифрование паролей пользователей
  • Создание различных прав использования приложения (Родительский, детский, главы семьи)
  • создание механизмов добавления зпрпботанных денег, вычитания потраченых денег
  • Создание системы перевда денег на чужой счет и возможность конфискации у более низких по знпчению пользователей

    • Инструменты для решения задачи:

  • Язык разметки HTML, XML
  • Языки PHP, JavaScript и Framework jQuery
  • СУБД MySQL

    • Пользователю при регистрации предоставляется аккаунт "Главы семи". Из этого аккауна можно создать родительские и детские аккаунты, с которыми можно в последствии зайти на сайт и пользоваться функционалом предоставляемым данному пользователю. Родительский аккаунт позволяет переводить\конфисковать деньгу у детских аккаунтов данной семьи. Так же всем аккаунтам предоставлена возможность тратить деньги. Все операции перевода, траты, добавления денег записываются в базу данных.
